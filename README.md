#  Odin Recipes — V2

## About this version
This second version of the project expands the original by introducing **CSS styling**, making the website more visually appealing and better structured.  
The update includes global styles, improved layout consistency, enhanced spacing, and a more polished presentation across all recipe pages.

---

##  About this project
Basic website created to practice HTML.  
This project consists of a main index page that contains links to several recipe pages.  
Each recipe page includes a description, an ingredients list, and step-by-step instructions.  

**New in V2:** Added CSS styling to improve layout, colors, spacing and readability.

---

## Sobre o projeto
Página web básica criada para praticar **HTML**  
O projeto consiste em uma página principal que contém links para algumas páginas de receitas.  
Cada página de receita inclui uma descrição, lista de ingredientes e passos detalhados.  

**Novidade da V2:** uso de CSS para melhorar o design, organização e aparência do site.
